# 記録アプリ（仮）

## サービス概要
学習の習慣化が苦手な人でも無理なく続けられるよう、  
「1日1件・1項目のみ」のシンプルな学習記録に特化したWebアプリです。

プログラミング学習や資格学習など、
日々の学習内容を最小限の手間で記録し、
振り返ることで学習の継続をサポートします。

## このサービスへの思い・作りたい理由
私は日々の習慣を続けるのが苦手で、「毎日何をしていたか分からないまま1日が終わる」ことに悩んでいました。  
既存の習慣管理アプリは入力項目が多く、次第に記録そのものが負担になってしまうことが多かったです。

そこで、  
「完璧に記録しなくていい」  
「とにかく少しでも残せればOK」  
という考え方で、習慣化が苦手な人でも使い続けられるアプリを作りたいと考えました。

## ユーザー層について
プログラミング学習や資格勉強をしている学生・社会人
学習記録をつけたいが、入力が面倒で続かなかった人
RUNTEQなどの自走型学習で、日々の学習を振り返りたい人

## サービスの利用イメージ
ユーザーは1日に1回、その日に行った学習内容を記録します。
記録内容は「今日何を学習したか」を一言で残すことを想定しています。

過去の学習記録は一覧で確認でき、
「どんな学習をどれくらい継続できているか」を
簡単に振り返ることができます。

詳細な管理ではなく、
「とにかく学習を続けること」を目的とした利用を想定しています。
例えば「RailsのCRUDを学習した」「SQLの復習をした」など、学習内容を一言で記録することを想定しています。

## ユーザーの獲得について
- SNSや個人ブログでの紹介
- 「習慣化が苦手な人向け」という切り口での発信
- 実体験をベースにした利用例の共有

## サービスの差別化ポイント・推しポイント
既存の学習記録サービスでは、
学習時間や教材、達成度など入力項目が多く、
記録自体が負担になりがちです。

本サービスでは、
「1日1件・1項目のみ」にあえて絞ることで、
学習記録のハードルを極限まで下げています。

「完璧な学習管理」ではなく、
「学習を続けるための最低限の記録」に特化している点が
最大の差別化ポイントです。

## 機能候補

- カテゴリ分け機能
- リマインダー通知
- 過去記録の検索機能
