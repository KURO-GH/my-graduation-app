# 記録アプリ（仮）

## サービス概要
習慣化が苦手な人でも無理なく続けられるよう、  
「1日1件・1項目のみ」のシンプルな記録に特化したWebアプリです。  
日々の行動を最小限の手間で記録し、振り返ることで習慣改善をサポートします。

## このサービスへの思い・作りたい理由
私は日々の習慣を続けるのが苦手で、「毎日何をしていたか分からないまま1日が終わる」ことに悩んでいました。  
既存の習慣管理アプリは入力項目が多く、次第に記録そのものが負担になってしまうことが多かったです。

そこで、  
「完璧に記録しなくていい」  
「とにかく少しでも残せればOK」  
という考え方で、習慣化が苦手な人でも使い続けられるアプリを作りたいと考えました。

## ユーザー層について
- 習慣を続けたいが、三日坊主になりがちな人
- 学習・運動・生活改善などをゆるく振り返りたい学生・社会人
- 多機能なアプリが苦手で、シンプルな操作を求めている人

## サービスの利用イメージ
ユーザーは1日に1回、その日に行ったことや意識したことを記録します。  
記録内容は一覧で確認でき、過去の自分の行動を簡単に振り返ることができます。  
「毎日完璧にやる」のではなく、「少しでも続ける」ことを目的とした利用を想定しています。

## ユーザーの獲得について
- SNSや個人ブログでの紹介
- 「習慣化が苦手な人向け」という切り口での発信
- 実体験をベースにした利用例の共有

## サービスの差別化ポイント・推しポイント
習慣化が苦手な人でも続けられるよう、  
**「1日1件・1項目のみ」のシンプルな記録に絞っている点**が最大の特徴です。

入力項目を最小限にすることで、
「書かなきゃいけない」という心理的な負担を減らし、  
自然と振り返りを続けられる設計にしています。

「多機能だから続かない」人に向けた、  
あえて“機能を絞った”習慣記録アプリです。

## 機能候補
### MVPリリース時
- ユーザー登録・ログイン
- 1日1件の記録作成
- 記録の一覧表示（日付順）
- 記録の編集・削除

### 本リリース時
- 習慣ごとの簡単な振り返り表示
- カテゴリ分け機能
- リマインダー通知
- 過去記録の検索機能

## 使用する技術スタック
- フレームワーク：Ruby on Rails
- データベース：PostgreSQL
- デプロイ：Render
