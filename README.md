# 記録アプリ（仮）

## サービス概要
日々の行動や習慣を記録し、あとから振り返ることで、
自分の生活を客観的に見直すことができるWebアプリです。
学習記録や生活習慣、作業ログなど、用途を限定せずに利用できます。

## このサービスへの思い・作りたい理由
日々をなんとなく過ごしてしまい、「結局何をしていたのか分からない」
と感じることが多くありました。
自分の行動を記録として残すことで、現状を客観的に把握し、
改善につなげられる仕組みが欲しいと考え、このサービスを作ろうと思いました。

## ユーザー層について
日々の生活や学習、作業内容を振り返りたいと考えている人を想定しています。
年齢や職業を問わず、自分の行動を見直したい人が気軽に使えることを目指します。

## サービスの利用イメージ
ユーザーはその日に行ったことを簡単に記録します。
一定期間が経つと過去の記録を一覧で確認でき、
自分の行動傾向や継続できている習慣を振り返ることができます。

## ユーザーの獲得について
SNSや知人への共有を通じて利用者を増やすことを想定しています。
「学習記録以外にも使える記録アプリ」として紹介することで、
幅広い利用シーンを想定しています。

## サービスの差別化ポイント・推しポイント
用途を限定せず、ユーザー自身が自由に記録内容を決められる点が特徴です。
学習・生活・作業など、1つのアプリで様々な記録をまとめて管理できます。

## 機能候補
### MVPリリース時
- ユーザー登録・ログイン機能
- 記録の投稿・編集・削除機能
- 記録の一覧表示

### 本リリース時
- 記録のカテゴリ分け機能
- 期間ごとの振り返り表示
- 簡単な統計・可視化機能

## 使用する技術スタック
- フレームワーク：Ruby on Rails
- データベース：PostgreSQL
- デプロイ：Render
