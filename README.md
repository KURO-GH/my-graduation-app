# 記録アプリ（仮）

## サービス概要
毎日の習慣を簡単に記録し、振り返ることで継続をサポートするWebアプリです。
学習や運動など、自分が改善したい習慣に特化しています。
習慣の登録・編集・削除と、過去の記録の一覧確認が可能です。

## このサービスへの思い・作りたい理由
私は日々の習慣を続けるのが苦手で、「毎日何をしていたか分からないまま1日が終わる」ことに悩んでいました。
自分の習慣を記録して振り返ることで、改善点や継続できたことを把握できるようになれば、生活の質を上げられると考えました。
その経験をもとに、「習慣の継続をサポートするWebアプリ」を作ろうと思いました。

## ユーザー層について
学生や社会人で習慣を継続したい人
日々の生活や学習・運動の改善に関心がある層を想定
習慣を簡単に振り返りたい人
シンプルなUIで記録と振り返りが簡単にできることで利用しやすい

## サービスの利用イメージ
ユーザーは毎日の習慣を登録・記録します。
過去の習慣記録をリストで振り返ります。
継続日数や達成状況を確認し、自分の習慣改善に活用します。
ユーザーは自分の行動を可視化することで、習慣を継続しやすくする。

## ユーザーの獲得について
SNSでの紹介や知人へのシェアを通じて利用者を増やす
「習慣の継続が苦手な人向け」として、個人ブログやTwitterで発信等

## サービスの差別化ポイント・推しポイント
単なる記録アプリではなく、習慣の継続に特化
過去の記録を簡単に振り返れるシンプルUI
原体験ベースで作られているため、機能や導線に説得力がある
MVP段階でも習慣の達成状況を可視化できる

## 機能候補
### MVPリリース時
ユーザー登録・ログイン
習慣の登録・編集・削除
日付ごとの記録一覧表示

### 本リリース時
習慣ごとの簡単な統計表示
カテゴリ別フィルタ
達成通知やリマインダー
過去ログの検索

## 使用する技術スタック
- フレームワーク：Ruby on Rails
- データベース：PostgreSQL
- デプロイ：Render
